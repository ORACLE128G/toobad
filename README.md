### 在线安装

- Google Chrome [插件地址](https://chrome.google.com/webstore/detail/too-bad/kgodpebcmnmfbdhfohdlggkakmkcplei)

### 手动安装

- 下载项目的zip文件，解压到本地

- Chrome右上角的设置按钮下找到更多工具，打开扩展程序

- 选择 加载已解压的扩展程序(如果没有显示先选中开发者模式)，选中解压后的文件夹，完成！
